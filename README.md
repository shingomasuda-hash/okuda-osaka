# 株式会社オクダ大阪 - 採用特設サイト

## プロジェクト概要
解体工事・産業廃棄物処理・リサイクル事業を展開する株式会社オクダ大阪の採用特設ランディングページです。

## 完成した主要機能

### ✅ 実装済み機能

1. **ヒーローセクション**
   - フルスクリーン背景画像（AI生成）
   - キャッチコピー「壊すんじゃない。次をつくる、最初の仕事。」
   - 2つのCTAボタン（応募・事業内容）
   - 会社概要バッジ（設立年・社員数・対応エリア）

2. **統計情報セクション**
   - 年間施工実績850件
   - リサイクル率98.5%
   - 連続無事故記録15年
   - 拠点数3箇所
   - カウントアップアニメーション付き

3. **事業内容セクション**
   - 総合解体工事（詳細な対応範囲）
   - 産業廃棄物処理（許可・処理フロー）
   - リサイクル事業（環境配慮の取り組み）
   - 各事業の特徴を箇条書きで詳細説明

4. **施工実績セクション**
   - 3つの代表的プロジェクト事例
   - 商業施設・集合住宅・工場の解体実績
   - 工期・リサイクル率の具体的数値
   - AI生成の現場画像

5. **社員インタビュー**
   - 3名の社員の声（現場監督・事務・重機オペレーター）
   - AI生成のポートレート画像
   - 入社年数・年齢・職種情報
   - リアルな体験談（600〜700文字）
   - ハッシュタグ形式のキーワード

6. **キャリアパスセクション**
   - 4段階の詳細なキャリアステップ
   - 入社〜3ヶ月：基礎研修
   - 4ヶ月〜1年：実務習得
   - 1年〜3年：中堅スタッフ
   - 3年以降：現場監督・管理職
   - 各段階で習得できるスキルを明記

7. **福利厚生セクション**
   - 9つの充実した福利厚生制度
   - 給与体系（月給・賞与・手当）
   - 資格取得支援制度
   - 安全への徹底投資
   - 休日・休暇制度
   - 健康管理サポート
   - 社内イベント
   - 通勤・社用車
   - 住宅サポート
   - 育児支援制度

8. **募集要項テーブル**
   - 3職種の詳細な募集内容
   - 応募資格・給与・勤務時間・休日
   - 待遇・福利厚生の詳細
   - 勤務地・選考プロセス

9. **FAQセクション**
   - 8つの詳細な質問と回答
   - アコーディオン式UI
   - 未経験者向け情報を充実

10. **応募方法セクション**
    - 3つの応募チャネル
    - メール・電話・LINE
    - 各チャネルの特徴説明

11. **レスポンシブデザイン**
    - PC（≥1024px）
    - タブレット（768-1023px）
    - スマホ（≤767px）
    - モバイル固定CTAボタン
    - ハンバーガーメニュー

12. **インタラクション**
    - スムーズスクロール
    - フェードインアニメーション（IntersectionObserver）
    - カウントアップアニメーション
    - FAQアコーディオン
    - カードホバーエフェクト
    - ヘッダースクロールエフェクト
